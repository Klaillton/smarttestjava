# smarttestjava
 Criação de aplicação JAVA Rest para teste da Smart Consulting
 
 The persistence was done in MariaDB whom is similar to MySQL but need their proper configurations who should be done in the file applications.properties, you can use any database you want, just configure this file and start the server.
 
 To access the application you could use the postman to have a better control about the JSON requisitions or access by the URL http://localhost:8080/difference/. You can use the "number" value to check the ocurrences. When using Postman you can use the POST method in the localhost:8080/difference so you can send the key "number" to register new values in the database, and check validations as well as messages to the user and developer.
